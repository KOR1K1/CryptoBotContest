# Frontend - Auction System

React frontend for the Auction System with real-time WebSocket updates.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs on http://localhost:3001

## Build

```bash
npm run build
```

## Environment Variables

- `VITE_API_URL` - Backend API URL (default: http://localhost:3000)
- `VITE_WS_URL` - WebSocket URL (default: http://localhost:3000)
